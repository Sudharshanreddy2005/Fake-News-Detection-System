# Dataset Instructions

Preferred dataset: **Kaggle - Fake and Real News Dataset**

1. Go to: https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset  
2. Download the dataset ZIP.
3. Extract and place these files in this `data/` folder:
   - `Fake.csv`
   - `True.csv`

Fallback option for quick testing:
- Use `sample_fake_news.csv` already included in this folder.
